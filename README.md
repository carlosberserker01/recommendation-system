# 🏠 Sistema de Recomendación de Propiedades

Aplicación web desarrollada con React + TypeScript que permite visualizar un listado de propiedades inmobiliarias y recibir recomendaciones de otras similares, basadas en criterios inteligentes.

El enfoque se centró en crear una aplicación funcional, modular, rápida y fácil de extender.

---

## 🚀 Tecnologías utilizadas

- React
- TypeScript
- Vite
- Tailwind CSS
- Zustand

---

## 📦 Instalación y uso local

```bash
git clone https://github.com/carlosberserker01/recommendation-system.git
cd recommendation-system

npm install

npm run dev
```

## ✅ Funcionalidades principales
🔎 Buscador por texto libre (normaliza tildes y mayúsculas)

🧠 Recomendaciones inteligentes según ciudad, tipo, precio, etc.

❤️ Favoritos persistentes usando Zustand y localStorage

✨ Scroll automático y resaltado visual al hacer clic en una recomendación
